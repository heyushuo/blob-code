1.引用组件按需加载,一些ui库也要按需加载(const search = resolve => require(['./search.vue'], resolve);) 2.循环调用子组件时添加 key (有利于vue快速定位那条数据发生了变化) 3.vue中使用keep-alive 4.vender包太大,可以使用cdn解决此问题,解决方法很简单，打包 vender 时不打包 vue、vuex、vue-router、axios 等，换用国内的 bootcdn 直接引入到根目录的 index.html 中。 5.组件销毁的时候去掉相应的监听事件 6.代码分割 分离业务代码和第三方库（ vendor ） 按需加载（利用 import() 语法） 7.提取组件的 CSS 到单独到文件 8.利用Object.freeze()提升性能 由于 Object.freeze() 会把对象冻结，所以比较适合展示类的场景
